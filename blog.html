Question-1: What are the different ways to select an element in the DOM?

we can select an element by using getElementById(), querySelector(), querySelectorAll(), getElementsByClassName(), or getElementsByTagName().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML is used to gets or sets HTML content inside an element, innerText is used to gets or sets only visible text inside an element and textContent is used to gets or sets all text content, including hidden elements.

Question-3: What is event delegation in the DOM?

Instead of adding events to multiple elements, you add one to a parent and let events bubble up.

Question-4: What is event bubbling in the DOM?

When an event on a child element also triggers the event on its parent and ancestors.

Question-5: How do you create, add, and remove elements using JavaScript?

By using document.createElement(), add with .appendChild() or .append(), and remove with .remove().
